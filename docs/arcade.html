About The Ar:Ca:De Project


Ar:Ca:De is an argumentation game library, of mini-games, that teach users argumentation skills and theory , through schemes implemented in argument cards or arg-cards. These skills, once applied, consequently teach users how to argue better.
Each of the mini-games within Ar:Ca:De act either as an individual games, or perform as levels when players opt for the more complex iterations of the games available, and skills are built up and carried across games


What does Ar:Ca:De stand for?


Ar:Ca:De stands for Argument Card Debate games- it comes from the names of the cards used within Ar:Ca:De and how they can be used in any number of games players can make up using argumentation principles.

The Aim behind Ar:Ca:De is that as people learn about different styles of arguments for themselves, they will also think more about what people are saying around them- especially online. This project forms part of the work done for the Edinburgh Napier Final year Honours project by Amantle Craviolatti.
