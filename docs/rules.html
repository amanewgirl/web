Shuffle Game Instructions


Did you know?
There are different styles of arguing, these are called Arguments. Arguments can be classified into different types, these are called Schemes

The Aim of the game:
The Aim of Shuffle is to sort the cards into the right order for the different argument style types or schemes given. Don't worry, its really easy!

How to Play

  1. Read the name of the argument scheme.Try to remember it, it might come in handy later!
  2. Read the cards one by one, you'll notice that each card is one person saying something in a conversation, and the other person is responding to the point they're making
  3. Read the example or argument description below the cards, try and see if you can pick up a pattern
  4. Read the cards again and when you're ready click the picture to shuffle the cards into the pattern you picked up earlier- the description is there to help you remember the pattern in case you forget
